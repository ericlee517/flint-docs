# 什么是点数？
Count（点数）是存在您ID中的一种虚拟货币，可用于一些消费场合。比如购买指定产品后，可获得部分点数。

点数是一种消费等级体现，若点数越多，ID等级越高：

| Count   | Level | Rights             |
| ------- | ----- | ------------------ |
| 0-59    | 1     | /                  |
| 60-109  | 2     | 铜色徽章            |
| 110-159 | 3     | /                  |
| 160-200 | 4     | /                  |
| 201-300 | 5     | 银色徽章            |
| 301-400 | 6     | /                  |
| 401-500 | 7     | /                  |
| 501-600 | 8     | 金色徽章            |
| 601-700 | 9     | /                  |
| 701-800 | 10    | /                  |
